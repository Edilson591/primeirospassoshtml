Historia dos computadores

O primeira geração de computadores da historia foi iniciada pelo Mark I criado em 1944, foi criado para criar cálculos, dessa desse mesmo período foi criado o Colossus em 1946, foi construida durante a guerra pela alemanha. Do outro lado da guerra foi criado o ENIAC no mesmo ano, esse por sua vez mais automatizado.

Depois de varias gerações de computadores chegamos aos dias de hoje, surgiu a Internet é um sistema global de redes de computadores interligadas que utilizam um conjunto próprio de protocolos (Internet Protocol Suite ou TCP/IP) com o propósito de servir progressivamente usuários no mundo inteiro. Pórem no inicio era apenas uma ideia de comunicação entre dois pontos.

O que siginifica Clients ?

O Clients é o dispositivo que consome uma informação da internet, que através dessa rede com computadores interligados consomem essa informação. Então funciona assim a informação vai através da internet, coleta informação dentro do servidor e retorna com a informação pedida. A forma que essa linguagem ser interpretada é pelo navegador.

Para que seja possível manipular a página do usuário diretamente, tomando ações dinâmicas que vão desde emitir uma mensagem, validação de um formulário, abrir e configurar janelas, manipulação de objetos da página, detectar versão do navegador. Um exemplo de validação de formulário que apresenta mensagens ao usuário quando os campos obrigatórios não são preenchidos. Assim, para situações que necessitam de interação com o usuário e não dependem do servidor, usamos a programação JavaScript.

A velocidade de conexão com a internet é diretamente proporcional ao resgate de recursos do servidor, então se o Conjunto de condigos for muito grande, pode demorar um pouco para carregar toda a pagina. além do tamanho do codigo o que pode dificultar na criação de um Site é a resolução, que deve ser considerada na criação de um website.

O que significa Servers ?

Um Servidor é um computador equipado com um ou mais processadores, bancos de memória, portas de comunicação, software e, às vezes, algum sistema de armazenamento de dados, como um disco rígido interno ou memória SSD. O termo servidor também define um recurso dentro de um sistema de computação maior capaz de processar aplicativos, fornecer serviços e armazenar dados. Esses sistemas podem ser físicos ou virtuais e podem ser instalados local ou remotamente. Normalmente, são escaláveis ​​e possuem alto poder de processamento.

O armazenamento dos dados do servidor e chamado Banco de Dados que funciona como uma organização e armazenamento de informações sobre um determinado domínio. De forma mais simples, é um agrupamento de dados que tratam de um mesmo assunto e precisam ser armazenados por segurança ou para reuniões futuras.

Dentro desse contexto de servidor e cliente tem um termo chamado Aplicações Web que foi uma solução criada que utliza a internet como meio de comunicação, não sendo preciso a instação. Conectando os clientes de forma conveniente e segura. As funções mais comuns do site, como carrinhos de compras, pesquisa e filtragem de produtos, mensagens instantâneas e feeds de notícias de mídia social, são projetadas para serem aplicativos da web. Eles permitem que você acesse funções complexas sem instalar ou configurar software.

Estrutura Básica do HTML

Para criar um site, um desenvolvedor precisa conhecer e entender a Estrutura Básica do HTML, ou seja, entender sua sintaxe e toda a estrutura burocrática de criação de elementos através das diversas tags existentes. Mesmo que o HTML tenha uma estrutura inicial simples, ficamos confusos na hora de montá-lo porque cada parte do código não é autoexplicativa e requer algum conhecimento prévio para isso. Porém, uma vez que você entenda as regras por trás da estrutura de cada linha do HTML, não precisa aprender de outra forma, pois temos apenas um modelo inicial de HTML. Continue conosco para saber mais sobre essa estrutura e todas as suas capacidades!

O HTML não é considerado uma linguagem de programação porque não pode criar funções dinâmicas. Em contraste, com HTML, os usuários podem criar e construir seções, parágrafos e links usando elementos, tags e atributos. Saiba mais sobre esta estrutura e todas as suas características, ou seja, o HTML auxilia na criação do codigo.

O uso mais comum do HTML são :

  • Desenvolvimento web: Os desenvolvedores usam códigos HTML para projetar como um navegador vai exibir os elementos das páginas, como textos, hiperlinks e arquivos de mídia.

  • Navegação na internet: Os usuários podem navegar facilmente e inserir links entre páginas e sites relacionados, já que o HTML é amplamente usado para incorporar hiperlinks.

  • Documentação: O HTML torna possível a organização e a formatação de documentos, de maneira similar ao Microsoft Word.

    Diferença entre listas ordenadas e não ordenadas

    A principal diferença entre uma lista ordenada e lista não-ordenada é como os elementos de cada lista são representados - em uma lista ordenada, os elementos são enumerados; Uma lista não ordenada coloca pontos antes de cada elemento.
